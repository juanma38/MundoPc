# Mundo Pc

Proyecto orden de compra componentes para armado de pc



## Installation

Install MundoPC with npm

```bash
  npm install MundoPC
  cd MundoPC
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/juanma38/MundoPc
```

Go to the project directory

```bash
  cd MundoPc
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
